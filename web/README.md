## Setup

Copy `.env.example` to `.env.local` in this folder and fill values:

```
NEXT_PUBLIC_SUPABASE_URL=...
NEXT_PUBLIC_SUPABASE_ANON_KEY=...
SUPABASE_SERVICE_ROLE_KEY=...
```

Install dependencies (root uses pnpm workspaces):

```
pnpm install
pnpm dev
```

Open http://localhost:3000
